UAlbany Thesis/Dissertation LaTeX Template
==========================================

A LaTeX template for theses and dissertations at the University at Albany, State University of New York.
It complies with the [Graduate School’s Formatting Guidelines](https://www.albany.edu/graduate/resources-current-students/dissertation-thesis-information), and is a streamlined alternative to the older template and document class available at the link above.
It is meant to be used with LuaLaTeX to produce accessible PDFs.
